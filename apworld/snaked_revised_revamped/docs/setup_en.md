# Snaked. Revised. Revamped. Setup Guide

Copy `apworld/snaked_revised_revamped` into an Archipelago checkout under
`worlds/snaked_revised_revamped`, or package it as a lowercase `.apworld` zip and place it in
Archipelago's `worlds` folder.

Phase 1 uses five locations and five score-bundle items. In the browser game, open Multiplayer,
choose Archipelago, enter the server address and slot name, then connect.
