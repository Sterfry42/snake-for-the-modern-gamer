## Overview

You are programming for the "Snake for the Modern Gamer" project. This program is written as a web-based game, using [Phaser](https://docs.phaser.io/phaser/getting-started/what-is-phaser) and [Vite](https://vite.dev/guide/). The primary languages are TypeScript and JavaScript. The program is a game designed to be ran on the web, and is based on the popular game "Snake".

The main entrypoint to the program is `src/main.js`, and many features are already implemented. You will likely be asked to implement new features or refactor old features. Please be careful not to regress any existing functionality.

## Rules

- Please understand the entire source code under `src/` before writing new features.
- Write new features in TypeScript or JavaScript depending on the files that need to be edited. If a new file is to be created for gameplay functionality, use TypeScript.
